# NeuralRetail — AI-Powered Retail Analytics Platform
NeuralRetail is an end-to-end Retail Analytics project built using Python, Machine Learning, and Streamlit. The platform simulates a real-world retail business environment with synthetic retail datasets and provides intelligent business insights through interactive dashboards.

The project combines:

Sales Analytics
Customer Segmentation (RFM Analysis)
Demand Forecasting
Churn Prediction
Inventory Optimization
Business Intelligence Dashboards

This project demonstrates practical Data Analytics, Machine Learning, Time Series Forecasting, and Dashboard Development skills used in modern retail and e-commerce industries.
